# ⚡ VoltGuard: Advanced Electrical Engineering Auditor

VoltGuard is a comprehensive, mathematical Power Systems Engineering application built on Django. It is designed to act as an automated consultant for industrial and residential electrical planning. By taking raw appliance inputs, the system instantly generates compliant Wiring Schedules, Hardware Bill of Materials (BoM), and Environmental Carbon impact predictions.

---

## 🚀 Core Features

### 1. The Predictive Analytics Engine (Real-World Feasibility)
VoltGuard goes beyond basic physics by instantly calculating the financial and environmental impact of a facility's power design.
* **Estimated Hardware BoM**: Automatically predicts the total cost (₹) of the physical installation by pricing out the exact copper wire gauges and MCB classifications required.
* **Monthly Electricity Forecast**: Calculates predicted energy consumption (kWh) based on customizable `Hours/Day` usage inputs, multiplying it by industrial tariffs to estimate the monthly utility bill.
* **Carbon Footprint Tracker**: Tracks the direct environmental impact by calculating the kilograms of $CO_2$ emissions generated by the facility's power demand.

### 2. Strict Power Systems Mathematics 
VoltGuard enforces 100% accurate, industry-standard formulas to ensure extreme safety:
* **The Power Triangle**: Converts Real Power (Watts) into true Apparent Power ($VA = W / P.F.$).
* **Demand Factor**: Aggregates the total connected load and applies a standard `0.8` Diversity Factor to calculate the true *Demand Limit* for the Main Incomer switch.
* **The 125% MCB Sizing Rule**: Ensures breakers never run at 100% continuous load ($I_{mcb} \ge I_{load} \times 1.25$). Automatically assigns rugged **Type C** breakers to inductive Motor/AC loads to survive inrush currents, and **Type B** to resistive lighting.
* **Dynamic Voltage Drops**: Utilizes standard solid Copper Ohms tables (e.g. 2.5mm² = 7.41$\Omega$/km). If the calculated voltage drop exceeds the strict $3.0\%$ international limit, VoltGuard automatically searches for and suggests visually upgrading the physical wire thickness.

### 3. The 3-Phase Auto-Balancer
If the facility's accumulated load breaches the standard 1-Phase limit (> 7kW), VoltGuard automatically upgrades the project to a 3-Phase system. It runs a descending greedy algorithm to intelligently distribute connected loads evenly across the L1, L2, and L3 lines, keeping neutral return currents as close to zero as possible.

### 4. Professional Exporting
VoltGuard features a `Print to PDF` generator. It renders a compliant, ready-to-deploy tabular schedule containing Wire Gauge sizes, Voltage Drop warnings, Amperage rules, and the designated Main Switch limits to hand directly to licensed contractors.

## 🛠️ Technology Stack
* **Backend:** Python, Django 5.x 
* **Database:** SQLite3 (Relational logic parsing)
* **Frontend:** HTML5, Tailwind CSS V3.4 (Glassmorphism & Advanced Gradients)
* **Export Engine:** ReportLab (Dynamic PDF Buffer rendering)

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mohansaina/finalyearproject.git
   cd finalyearproject/voltguard
   ```

2. **Activate the Virtual Environment:**
   ```bash
   # Windows
   .\venv\Scripts\activate
   ```

3. **Install Dependencies:**
   ```bash
   pip install django reportlab
   ```

4. **Run Database Migrations:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

5. **Start the Development Server:**
   ```bash
   python manage.py runserver
   ```
   Navigate to `http://127.0.0.1:8000/` in your browser.
